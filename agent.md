# tanstack-router
routeTree.gen.ts never needs to be modified, even with relating ts errors (for example when a new route is created). tanstack-router will always update routeTree.gen.ts automatically.

# utils
every file in the utils folder show have 1 utils function only. the filename should be the same as the function name.

# imports
always use the basepath "@" for imports, never use relative imports.
